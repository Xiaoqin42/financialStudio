# AlphaAnalyst 用户使用手册
快速入门与功能指南

## 1. 账户管理

### 注册与登录
首次使用请点击左侧栏的 **"Sign In"** 按钮。
- 选择 "Create Account" 进行注册。
- 输入邮箱并点击 "Send Verification Code"（演示模式下验证码会直接显示在弹窗中）。
- 设置强密码（需包含字母、数字及特殊字符，且不少于8位）。

### 个人资料与多账户
登录后点击左下角的用户卡片可修改头像或昵称。
系统支持多账户切换，每个账户的搜索历史、头像和设置完全独立。退出登录后，界面将重置为初始状态，保护您的隐私。

## 2. 智能市场分析
在主页搜索框输入任何公司名称或股票代码（如 "AAPL", "Tesla", "0700.HK"），AI 将为您生成深度报告。

- **财务摘要**: 获取公司业务范围、市盈率、市值等关键财务指标的即时概览。
- **AI 预测**: 基于大模型的趋势预测，包含目标价格、看涨/看跌倾向及详细的推理逻辑。

## 3. 交互式图表与指标
分析页面底部的图表区提供模拟的实时市场数据。您可以通过点击图表下方的标签来开启或关闭特定指标。

| 指标名称 | 功能说明 |
|----------|----------|
| **MA5 / MA20** | 移动平均线。MA5 反映短期趋势，MA20 反映中期支撑/阻力位。 |
| **RSI (相对强弱指数)** | 衡量价格变动速度。超过 70 通常表示超买（可能回调），低于 30 表示超卖（可能反弹）。 |
| **Bollinger Bands (布林带)** | **新增功能！** 包含上轨和下轨。价格触及上轨可能表示超买，触及下轨可能表示超卖。图表中蓝色阴影区域代表价格波动的正常范围。 |
| **Stoch (随机指标)** | 比较收盘价与价格范围，用于识别趋势反转的潜在动能。 |

> **提示**：将鼠标悬停在图表上的数据点，可以看到详细的每日价格、成交量及所有激活指标的具体数值。

## 4. 常见问题 (FAQ)

**Q: 我的搜索历史会保存多久？**
A: 搜索历史永久保存在您当前的浏览器中，直到您手动清除浏览器缓存或在设置中删除账户。

**Q: 图表数据是真实的吗？**
A: 本应用中的 AI 文本分析（如市盈率、风险因素）基于真实的网络搜索数据。但为了演示交互性，**K线图表数据目前是基于算法生成的模拟数据**，仅用于展示技术指标的可视化效果。
